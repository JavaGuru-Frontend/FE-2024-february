    box-sizing: border-box; - Применяется для изменения алгоритма расчёта ширины и высоты элемента.
    Описание

    Блочным элементам можно задавать всё отступы - и padding и margin с 4 сторон

    Строчные элементы занимают столько места, сколько нужно для контента - нельзя задать ширину или margin сверху и снизу

    Элементам в css можно изменить тип, для того чтобы их выравнивать и задавать отступы, например строчный элемент можно сделать строчно-блочным, для этого в css надо задать display: inline block - https://cssreference.io/property/display/
